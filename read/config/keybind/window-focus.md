# 視窗聚焦切換


* [設定片段](https://github.com/samwhelp/note-about-openbox/tree/gh-pages/_demo/config/openbox-config/main/share/gen/openbox-gen-rc/Section/Keybind/WindowSwitch.php)


## 前後聚焦切換


| 按鍵組合  | 功能                   | 執行指令               |
| ----------| ---------------------- | ---------------------- |
| `Win + a` | 聚焦切換到前面一個視窗 | `PreviousWindow` (openbox 內建) |
| `Win + s` | 聚焦切換到後面一個視窗 | `NextWindow` (openbox 內建) |


> 也可以在「Panel」的「視窗區」，使用「滑鼠中鍵」，上下滾動，循環切換「視窗」。
