## Default Config

run

``` sh
ls -1 /etc/xdg/openbox/
```

show

```
autostart
environment
menu.xml
rc.xml
```

## Default Config / Copy Command

run

``` sh
cp -rf /etc/xdg/openbox/. ./
```

or run

``` sh
cp -rf /etc/xdg/openbox/* ./
```
